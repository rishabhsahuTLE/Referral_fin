# Referral Payout Dashboard

Enhanced referral payout management system with university switching, multi-tab filtering, and stage-based tracking.

## Features

- University switcher (IIT Bhilai, IIT Mandi, Jama Hamdard)
- Multi-tab view: Urgent, All, Stages, Doc Verification, Bank Verification, Payment Pending
- Color-coded status stages
- Urgent item highlighting (red)
- Description with flag option for each row
- Payment history tracking
- Monthly reports with course-wise analytics

## Getting Started

```bash
npm install
npm start
```

The app will run on `http://localhost:3000`
